介绍：CC-Mall-b2c 商户
===============

使用工具 

thinkphp6.x ，引入（路由、模板、多应用、多语言、验证码）模块，并采用强路由模式

layui

vue2.x + elementui2.x （cdn外部引入，非npm安装）



运行
===============

初次 git pull 代码下来，

1.请先进入根目录运行 

```php
composer install 
或者 
composer update 
安装依赖
```



2.运行项目

```php
php think run
```

访问地址：http://localhost:8000/manager/login
